# Monad Native Day 1 Toolkit

Build your personalized Monad Mainnet Day 1 Toolkit. Select your favorite DeFi protocols, wallets, and tools to prepare for Monad's launch.

- **11 Categories**: Choose from DeFi, DEX, Lending, Stablecoins, Bridges, Wallets, NFT Marketplaces, Oracles, Gaming, Social, and Launchpads

## ðŸš€ Getting Started

### Prerequisites

This project requires **Node.js** (v16 or higher). We recommend using **NVM (Node Version Manager)** to install and manage Node.js versions.

#### Installing NVM

##### macOS / Linux

1. Install NVM using the install script:

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
```

Or using wget:

```bash
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
```

2. Add NVM to your shell profile (if not done automatically):

```bash
# For bash
echo 'export NVM_DIR="$HOME/.nvm"' >> ~/.bashrc
echo '[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"' >> ~/.bashrc
source ~/.bashrc

# For zsh
echo 'export NVM_DIR="$HOME/.nvm"' >> ~/.zshrc
echo '[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"' >> ~/.zshrc
source ~/.zshrc
```

3. Verify installation:

```bash
nvm --version
```

4. Install Node.js:

```bash
nvm install --lts
nvm use --lts
```

##### Windows

**Option 1: nvm-windows (Recommended)**

1. Download the installer from [nvm-windows releases](https://github.com/coreybutler/nvm-windows/releases)
2. Download `nvm-setup.exe` from the latest release
3. Run the installer and follow the setup wizard
4. Open a new Command Prompt or PowerShell as Administrator
5. Verify installation:

```cmd
nvm version
```

6. Install Node.js:

```cmd
nvm install lts
nvm use lts
```

**Option 2: Direct Node.js Installation**

If you prefer not to use NVM on Windows, download Node.js directly from [nodejs.org](https://nodejs.org/) and run the installer.

#### Verify Node.js Installation

After installing Node.js (via NVM or direct installation), verify it's working:

```bash
node --version
npm --version
```

You should see version numbers for both Node.js and npm.

### Local Development

1. Clone the repository:

```bash
git clone https://github.com/nafyn/neverland-monadtoolkit.git
cd neverland-monadtoolkit
```

2. Install dependencies:

```bash
npm install
```

3. Build Tailwind CSS:

```bash
npm run build:css
```

Or watch for changes during development:

```bash
npm run watch:css
```

4. Start the development server:

```bash
npm start
```

This will start a local server on `http://localhost:8080` using [serve](https://github.com/vercel/serve) (by Vercel).

### Deployment

The project is deployed on Vercel at:

```
https://toolkit.neverland.money/
```

**Important for Production:**

This project uses **Tailwind CSS build process** (not the CDN) for optimal performance:

- **Why?** Tailwind CDN (~50KB) is render-blocking and not recommended for production
- **Solution:** We build a minified CSS file (~19KB) containing only the classes we use
- **Build:** Vercel automatically runs `npm run build:css` on every deployment

If you deploy elsewhere, ensure your deployment platform runs the build command:

```bash
npm install
npm run build:css
```
